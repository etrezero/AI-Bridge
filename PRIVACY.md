# Privacy Policy for AI-Bridge GPT

**Last Updated: April 1, 2026**

## Overview
This GPT (AI-Bridge) connects to the GitHub repository (etrezero/AI-Bridge) to execute commands and retrieve results.

## Data Collection
- This GPT does not collect personal data
- API calls are made to GitHub API only
- No data is stored outside of the GitHub repository

## Data Usage
- Commands are stored in gpts_commands/ folder
- Results are stored in gpts_results/ folder
- All data remains within your GitHub repository
